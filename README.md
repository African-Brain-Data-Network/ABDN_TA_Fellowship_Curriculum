# ABDN TA Curriculum Repository

## Overview

This repository contains structured teaching curricula for multiple neuroimaging and neuroscience modalities, developed for the **ABDN Teaching Assistant (TA) Program**. The goal is to provide a modular, reproducible, and practical learning pathway that moves from foundational theory to hands-on data analysis and scientific writing.

The curriculum is organized into four core modalities:

* EEG (Electroencephalography)
* MRI / fMRI (Magnetic Resonance Imaging / Functional MRI)
* Electrophysiology
* fNIRS (Functional Near-Infrared Spectroscopy)

Each modality is delivered over a **5-month period (June–October)** and follows a consistent progression:

1. Foundations and theory
2. Data acquisition and experimental design
3. Data analysis and tools
4. Scientific writing
5. Final paper development and presentation

## Curriculum Design Philosophy

This curriculum is designed around three key principles:

### 1. Progressive Learning

Students begin with **fundamental neuroscience concepts**, gradually moving toward:

* Experimental design
* Data acquisition
* Signal processing
* Real-world dataset analysis

### 2. Hands-On Experience

Each track emphasizes practical engagement:

* Use of real or simulated datasets
* Exposure to standard research tools (e.g., Python-based workflows, domain-specific software)
* Guided analysis exercises

### 3. Research-Oriented Output

By the end of the program, students are expected to:

* Conduct literature reviews
* Analyze domain-specific data
* Write and refine a scientific paper
* Present their findings

---

## Monthly Breakdown (General Framework)

### June — Foundations

* Core neuroscience concepts
* Modality-specific principles
* Biological and physical underpinnings

### July — Data Acquisition & Experimental Design

* Instrumentation and setup
* Signal quality considerations
* Ethics in human and animal research
* Designing reproducible experiments

### August — Data Analysis & Tools

* Introduction to analysis pipelines
* Software tools and workflows
* Signal processing and statistical methods
* Practical exercises with datasets

### September — Scientific Writing

* Topic selection and scoping
* Literature review techniques
* Structuring a scientific paper
* Drafting key sections (Introduction, Methods, Results, Discussion)

### October — Finalization & Presentation

* Editing and peer review
* Final paper submission
* Oral or poster presentation preparation

---

## Target Audience

This curriculum is intended for:

* Undergraduate and graduate students in neuroscience, biomedical engineering, psychology, or related fields
* Early-stage researchers seeking structured exposure to neuroimaging methods
* Teaching assistants facilitating lab-based or discussion-based learning

---

## Learning Outcomes

By the end of the program, participants should be able to:

* Explain core principles of each neuroimaging modality
* Design basic experiments and data collection protocols
* Perform introductory data analysis using appropriate tools
* Interpret results within a scientific framework
* Write and present a structured research paper

---

## How to Use This Repository

### For Teaching Assistants

* Follow each weekly structure as a guide for lectures or sessions
* Adapt pacing depending on student background
* Incorporate additional datasets or case studies where possible

### For Students

* Use each weekly topic as a study module
* Complete suggested activities and explore tools independently
* Collaborate during discussion and writing phases

### For Program Coordinators

* Modify timelines to fit institutional schedules
* Integrate assessments, grading rubrics, and checkpoints
* Expand modules with guest lectures or workshops

## Suggested Tools & Resources

While not strictly required, the following tools are recommended:

* Python (NumPy, SciPy, Matplotlib)
* MNE-Python (EEG analysis)
* EEGLAB (MATLAB-based EEG toolbox)
* SPM / FSL (fMRI analysis)
* Data visualization tools
